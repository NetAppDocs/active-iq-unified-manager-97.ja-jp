= レポートのカスタマイズ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
さまざまな方法でビューをカスタマイズして、 ONTAP クラスタの管理に必要なすべての情報を含むレポートを作成できます。

デフォルトのインベントリページまたはカスタムビューで開始し、列の追加や削除、列の順序変更、データのフィルタリング、特定の列での昇順または降順のソートなどを行ってカスタマイズします。

[NOTE]
====
レポートを管理するには、アプリケーション管理者またはストレージ管理者のロールが必要です。

====
