= SVMの関連付けの表示
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Storage VM Associationsページを使用すると、既存のSVMの関連付けとそのプロパティを表示したり、追加のSVMの関連付けが必要かどうかを確認したりできます。



== 作業を開始する前に



== 手順

. 左側のナビゲーションペインで、* Protection *>* SVM Associations *をクリックします。
+
SVMの関連付けとそのプロパティのリストが表示されます。
